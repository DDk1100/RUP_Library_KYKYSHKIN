# Глоссарий проекта
| Термин | Что означает |
|--------|----------------|
| Читатель | Студент или сотрудник техникума, который берёт книги |
| Книга | Издание, которое есть в библиотеке (может быть в нескольких экземплярах) |
| Экземплер | Конкретный физический книжный экземпляр |
| Бронь | Временное закрепление книг за четателем (обычно на 2 дня) |
| Выдача | Передача книги читателю на срок |
| Задолжнность | Просрочка возврата книг |
| Каталог | Список всех книг с возможностью поиска |
